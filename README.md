# TrackPro - Advanced Parcel Tracking System

## Overview
TrackPro is a modern web-based parcel tracking system that allows users to track their deliveries in real-time. Built with HTML, CSS, and JavaScript, it provides a seamless user experience for monitoring package status, location, and delivery estimates.

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git (for cloning the repository)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/wolfie2007/trackpro.git
cd trackpro
```

2. Open in your browser:
```bash
# Simply open index.html in your preferred browser
# Or use a local server (recommended)
python -m http.server 8000
# Then navigate to http://localhost:8000
```

## Usage

### Tracking a Parcel
1. Navigate to the "Track Parcel" section
2. Enter your tracking number (format: TRK123456789)
3. Click "Track Now"
4. View detailed package information, timeline, and route map
5. Customize notification preferences as needed

### User Authentication
- Visit the **Login** page for existing users
- Visit the **Sign Up** page to create a new account
